Política de Privacidade do Wild Cards Game

Nós, da FApps Fun, desenvolvemos o Wild Cards Game (doravante mencionado como WCG) como um jogo gratuito para jogar, mas suportado por propagandas e compras no jogo opcionais. Esta página é usada para informar você sobre as políticas de coleta, uso e divulgação de informações pessoais, se você decidir jogar o WCG. Ao instalar e abrir o jogo, você declara concordar com todos os termos descritos nesta política.

1. Coleta e uso de informações

Para uma melhor experiência, posso coletar dados de registro durante o jogo (descritos na seção 2). As informações que eu solicitar serão retidas no seu dispositivo e no banco de dados do Google Firebase, para acesso remoto por administradores apenas para garantir o correto funcionamento do jogo. Não há nenhuma coleta de dados pessoais do usuário e nenhum dado é vendido.

O WCG utiliza serviços de terceiros que podem coletar informações usadas para identificá-lo. O link para a política de privacidade de provedores de serviços terceirizados usados ​​pelo aplicativo estão listados a seguir:

  • Google Play Services
  • AdMob
  • Google Analytics for Firebase
  • Firebase Crashlytics
  • OpenAI API

2. Compras no Jogo

Para proporcionar uma experiência ainda mais imersiva e personalizada no WCG, posso oferecer a oportunidade de realizar compras no aplicativo. Todas as compras são realizadas através do Google Play, o que garante um processo seguro. A seguir, explico como essas compras funcionam:

2.1 Compras únicas

As compras únicas permitem ao jogador adquirir recursos sem renovação automática. Os principais itens incluem:

  • Remoção de anúncios: uma compra única que remove propagandas dentro do jogo.
  • Pacote de criação de cartas: permite aos jogadores criar e enviar suas próprias cartas.

2.2 Assinaturas

O WCG oferece uma assinatura completa, que inclui a remoção de anúncios e acesso a todos os pacotes de cartas, além de um narrador IA para uma experiência mais imersiva.

3. Cartas do Cards Against Humanity

Incorporamos cartas do Cards Against Humanity sob a licença Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International (CC BY-NC-SA 4.0).

4. Dados de registro

Durante o jogo, podem ser coletados "Dados de registro", como o endereço IP e outros dados que ajudam a identificar problemas técnicos.
